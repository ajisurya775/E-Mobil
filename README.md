## Cara Intalasi

-   Pertama create database dengan nama e-mobil.
-   Kedua buka code editor kemudian koneksikan dengan database yang telah dibuat
-   Ketiga buka terminal pindah ke directory dan jalankan perintah php artisan migrate --seed.
-   Terakhir jalankan perintah php atisan serve untuk menjalankan project
